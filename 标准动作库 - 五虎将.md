## 一、马龙 - 六边形战士

### 1.1 选手简介
- **出生日期**: 1988-10-20
- **握拍**: 右手横拍
- **打法**: 弧圈结合快攻
- **特点**: 技术全面、无明显短板、稳定性极强

### 1.2 正手攻球（标准动作）

**动作要领**:
```
准备姿势:
- 双脚开立与肩同宽，左脚稍前
- 膝盖微屈，重心降低
- 身体稍向右转，右肩下沉
- 持拍手自然置于身体右侧

引拍阶段:
- 转腰带动手臂向后引拍
- 肘部自然弯曲约 90-100 度
- 拍面稍前倾（约 75-80 度）
- 重心移至右脚

击球阶段:
- 蹬地转腰，重心从右脚移至左脚
- 前臂快速收缩，手腕辅助发力
- 击球中上部，摩擦球
- 击球点：身体右前方约 30cm

随挥阶段:
- 手臂自然挥至左前方
- 拍面指向击球方向
- 重心完全移至左脚
- 迅速还原准备姿势
```

**关键角度**:
| 部位 | 标准角度 | 允许范围 |
|------|----------|----------|
| 肘部 | 95° | 85-105° |
| 膝部 | 120° | 110-130° |
| 躯干 | 10°前倾 | 5-15° |
| 拍面 | 75° | 70-80° |

**常见错误**:
1. ❌ 肘部过度外展 → 影响稳定性
2. ❌ 只用手臂不用腰 → 力量不足
3. ❌ 击球点过晚 → 失去最佳发力时机

### 1.3 反手推挡

**动作要领**:
```
- 肘部贴近身体右侧
- 前臂外旋引拍
- 击球时前臂内旋发力
- 击球中上部，向前上方摩擦
- 击球点：身体正前方
```

### 1.4 发球（侧旋）

**动作要领**:
```
- 左脚在前，身体侧对球台
- 持球手抛球至 16cm 以上
- 手腕快速抖动，摩擦球右侧
- 第一落点在己方球台中部
```
