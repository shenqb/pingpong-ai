# 标准动作库

## 图片使用说明

将乒乓球标准动作图片放在此目录，命名规则：

- `forehand.jpg` - 正手攻球
- `backhand.jpg` - 反手推挡
- `forehand_loop.jpg` - 正手拉弧圈
- `backhand_loop.jpg` - 反手拉弧圈
- `serve.jpg` - 发球
- `flick.jpg` - 挑打

## 图片要求

1. **侧面视角** - 最佳分析角度
2. **全身入镜** - 包含完整身体姿态
3. **清晰可见** - 关键部位不遮挡
4. **专业选手** - 马龙、张继科、许昕等

## 图片来源

建议从以下渠道获取：

1. 中国乒乓球协会官网
2. 国际乒联(ITTF)官网
3. 专业乒乓球教学视频截图
4. 体育教材扫描
