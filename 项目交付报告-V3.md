## 📞 使用说明

### 本地开发

```bash
# 1. 克隆项目
git clone https://github.com/shenqb/pingpong-ai.git
cd pingpong-ai

# 2. 安装依赖
cd backend && npm install
cd ../frontend && npm install

# 3. 初始化数据库
cd database && python3 init-db.py

# 4. 启动服务
# 终端 1: cd backend && npm run dev
# 终端 2: cd frontend && npm run dev

# 5. 访问 http://localhost:3000
```

### 原型预览

**直接访问**:
```
https://skirt-arnold-referred-equally.trycloudflare.com
```

或本地打开:
```bash

```
