## 🎯 功能描述

### 功能列表

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P0 | 视频录制 | 调用摄像头录制视频 |
| P0 | 实时分析 | MediaPipe 客户端分析 |
| P0 | 结果显示 | 实时显示分数和关键点 |
| P1 | 视频回放 | 录制完成后回放 |
| P1 | 结果保存 | 保存分析记录 |

### 用户故事

```
作为用户，我希望能够录制我的打球视频并实时获得分析，
这样我可以立即了解动作问题并调整。

验收标准:
- 录制视频时实时显示姿态关键点
- 延迟 < 500ms
- 显示当前分数
- 录制完成后可查看详细分析
```

## 🔧 技术方案

### 技术选型

| 组件 | 技术 | 说明 |
|------|------|------|
| 视频录制 | WebRTC | 浏览器原生支持 |
| 姿态检测 | MediaPipe Pose | 客户端分析 |
| 渲染 | Canvas | 骨架绘制 |
| 性能 | WebWorker | 后台处理 |

### 实现方案

**前端流程**:
```
1. 调用摄像头
2. 获取视频流
3. MediaPipe 分析每帧
4. Canvas 绘制骨架
5. 实时显示分数
6. 录制完成保存
```

**性能优化**:
- 降低视频分辨率（720p）
- 降低分析帧率（15fps）
- WebWorker 后台处理
- 结果缓存
