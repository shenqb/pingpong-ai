## 🚀 快速开始

### 方式一：直接打开

```bash
# 在浏览器中打开
open index.html
```

或双击 `index.html` 文件

### 方式二：本地服务器

```bash
# 使用 Python
python3 -m http.server 8080

# 使用 Node.js
npx http-server

# 访问 http://localhost:8080
```

## 🔧 自定义

### 修改配色

编辑 `styles.css` 中的 CSS 变量：

```css
:root {
    --primary-blue: #你的颜色;
    --accent-orange: #你的颜色;
}
```

### 修改尺寸

```css
.phone-frame {
    width: 你的宽度;
    min-height: 你的高度;
}
```

### 添加页面

1. 在 `index.html` 中添加新的 `.page` div
2. 在原型导航中添加对应的按钮
3. 添加页面内容
