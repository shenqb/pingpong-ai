# 🏓 PingPong AI - UI 原型设计 V3.0

> **设计版本**: V3.0  
> **设计日期**: 2026-03-25  
> **设计参考**: SwingVision + HomeCourt + Keep  
> **设计风格**: 现代运动科技风

---

## 一、设计理念

### 1.1 设计原则

| 原则 | 说明 | 参考 |
|------|------|------|
| **专业感** | 运动科技的专业视觉，建立信任 | SwingVision |
| **活力感** | 鲜明的色彩和动效，激发运动热情 | HomeCourt |
| **易用性** | 清晰的层级和引导，降低学习成本 | Keep |
| **数据可视化** | 直观的数据展示，让用户看到进步 | 三者结合 |

### 1.2 设计关键词

```
专业 · 动感 · 清晰 · 现代 · 科技感
```

## 二、视觉系统

### 2.1 配色方案

#### 主色调

| 颜色 | 色值 | 用途 |
|------|------|------|
| **科技蓝** | `#0066FF` | 主按钮、重要操作、品牌色 |
| **活力橙** | `#FF6B35` | 强调色、CTA 按钮、成就 |
| **成功绿** | `#00C853` | 成功状态、进步指示 |
| **警示红** | `#FF3B30` | 错误、需改进项 |

#### 中性色

| 颜色 | 色值 | 用途 |
|------|------|------|
| **深空黑** | `#0D1117` | 主背景（深色模式） |
| **碳素灰** | `#161B22` | 卡片背景 |
| **中灰** | `#8B949E` | 次要文本 |
| **浅灰** | `#F0F6FC` | 主背景（浅色模式） |
| **纯白** | `#FFFFFF` | 卡片背景（浅色模式） |

#### 渐变色

```css
/* 主渐变 - 用于 Hero 区域、重要卡片 */
--gradient-primary: linear-gradient(135deg, #0066FF 0%, #00C6FF 100%);

/* 成功渐变 - 用于高分、进步 */
--gradient-success: linear-gradient(135deg, #00C853 0%, #69F0AE 100%);

/* 活力渐变 - 用于 CTA、推荐 */
--gradient-vibrant: linear-gradient(135deg, #FF6B35 0%, #FF8E53 100%);

/* 深色卡片渐变 */
--gradient-card-dark: linear-gradient(180deg, #161B22 0%, #0D1117 100%);
```

### 3.1 首页 (Home)

#### 布局结构

```
┌─────────────────────────────────┐
│  状态栏                          │
├─────────────────────────────────┤
│  ┌───────────────────────────┐  │
│  │   Hero 区域               │  │
│  │   - 品牌 Logo + 标语      │  │
│  │   - 今日统计卡片          │  │

│                                  │
│  ┌───────────────────────────┐  │
│  │   快捷入口 (3 列)          │  │
│  │   [动作分析][标准动作]    │  │
│  │   [实时分析]              │  │
│  └───────────────────────────┘  │
│                                  │
│  ┌───────────────────────────┐  │
│  │   最近分析 (列表)         │  │
│  │   - 分数 + 动作名 + 趋势  │  │
│  └───────────────────────────┘  │
│                                  │
│  ┌───────────────────────────┐  │
│  │   推荐训练 (横向滚动)     │  │
│  └───────────────────────────┘  │
├─────────────────────────────────┤

│  [首页][分析][+][课程][我的]   │
└─────────────────────────────────┘
```

#### 视觉设计

**Hero 区域**:
- 背景：主渐变 (科技蓝)
- Logo: 白色大字号 + 乒乓球图标
- 统计卡片：毛玻璃效果 + 白色文字
- 动效：数字滚动动画

**快捷入口**:
- 3 列卡片布局
- 每卡片：图标 + 标题 + 描述 + 箭头
- 图标：渐变色圆形背景
- 点击：轻微缩放反馈

**最近分析**:
- 列表项：左分数 + 中信息 + 右趋势
- 分数：圆形徽章，颜色根据分数变化
- 趋势：↑绿色 / ↓红色 / →灰色

## 四、交互设计规范

### 4.1 动画规范

| 场景 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 页面切换 | 从右滑入 | 300ms | ease-out |

- 浅灰色背景

- 圆形旋转
- 品牌色
- 居中显示

**无数据**:
- 大号插图
- 说明文字
- 引导操作按钮
